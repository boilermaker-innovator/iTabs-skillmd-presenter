ü§î What is this?

Claude Skills are folders with instructions that teach AI how to do specific tasks. They live as SKILL.md files on GitHub.

Problem: Raw SKILL.md files are ugly and hard to read.

Solution: This tool converts them into beautiful, expandable iTabs pages that anyone can understand.

üìã How to use

Option 1: Paste from GitHub

  • Find a SKILL.md on GitHub
  • Paste the URL and click "Fetch"
  • Click "Convert to iTabs"

Option 2: Paste content directly

  • Copy the raw SKILL.md text
  • Paste into the text area
  • Click "Convert to iTabs"

Or click "Load Example" to try it with a sample skill.

üìÑ What's a SKILL.md?

A simple text file with two parts:

1. Frontmatter (the label):

---
name: my-skill
description: What it does
---

2. Instructions (the knowledge):

Written in plain English with headings and steps that Claude follows.

  • Skill creators: Make your GitHub skills look professional
  • Teams: Create browsable docs for internal skills
  • Discovery: Understand what a skill does before installing
  • Sharing: Send someone a nice preview instead of raw code
